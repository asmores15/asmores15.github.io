The reason I chose this project is because I was reading *Code: The Hidden Language of Computer Hardware and Software*
and it talks about the Intel 8080 microprocessor and the basics of how it works.
I think CPU architecture is super interesting, so I decided to give it a shot!

I'll be uploading my files to a [repository](https://github.com/asmores15/eightyeighty) on my GitHub profile. My first step will be the ALU, I think. 
I thought about just going all in and doing every component at once but I think I'd get lost in the weeds pretty quickly.
So one step at a time for me :)

**Components Checklist**
- ALU
- Decimal Adjust
- Instruction Decoder and Machine Cycle Encoder
- Data Bus Buffer/Latch
- Register Array
- Address Buffer
- Instruction Register
- Timing and Control

That's the list I'm going to start with, it might change in the future depending on what happens.
I'm basing it off of this block diagram from the 8080 data sheet

![8080 block diagram](/images/8080_block_diagram.png)
